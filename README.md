# Rising FC Website

This is the official website for **Rising FC**, a futsal team.<br>
LINK-->https://rising-fc.web.app/

## Features
- About Us section
- Team player profiles
- Match schedule
- Merchandise shop
- Join Team form
- Contact form

## How to use
- Clone or download the repo
- Open `index.html` in your browser

## Folder structure
- `index.html`
- `images/` (all images used on the site)

## Tech
- HTML
- CSS
- JavaScript
